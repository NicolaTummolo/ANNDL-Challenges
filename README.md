# ANNDL Challenges

This repository contains the material developed for the two challenges of the **Artificial Neural Networks and Deep Learning** course at Politecnico di Milano.

The repository includes the final notebooks and reports submitted for the assignments.


## Challenge 1

In the first assignment, we received a collection of images, each coming from a low-magnification Whole Slide Image (WSI) of human tissue. The objective was to perform image classification by assigning each image to one of four possible classes, corresponding to molecular subtypes identifying a potential disease within the tissue.

Our solution included data preprocessing, model design, training, validation, and final evaluation.
The project was ranked in the **top 10% according to the professors' evaluation**.

Materials:

* [Notebook](Challenge-1/Challenge_1.ipynb)
* [Report](Challenge-1/ANNDL_report_challenge_1.pdf)

## Challenge 2

The second assignment focused on another deep learning classification task. Starting from the provided dataset, we designed and evaluated a complete neural network pipeline, including preprocessing, training strategy, model selection, and final assessment.
