# ANNDL Challenges

This repository contains the material developed for the two challenges of the **Artificial Neural Networks and Deep Learning** course at Politecnico di Milano.

The repository includes the final notebooks and reports submitted for the assignments.


## Challenge 1 - Time Series Classification 

In this assignment, we received a collection of multivariate time series. Each sequence consists of 180 time steps and contains measurements from several input channels. The objective of the task was to perform time series classification, assigning each entire sequence to one of three possible classes. Our solution included data preprocessing, model design, training, validation, and final evaluation. The project was ranked in the **top 10% according to the professors' evaluation**.

Materials:

* [Notebook](Challenge-1/Challenge_1.ipynb)
* [Report](Challenge-1/ANNDL_report_challenge_1.pdf)

## Challenge 2 - Image Classification 

In this assignment, we received a collection of images. Each image comes from a low-magnification Whole Slide Image (WSI) of human tissue. The objective of the task was to perform image classification, assigning each image to one of four possible classes corresponding to molecular subtypes identifying a potential disease within the tissue. Our solution included image preprocessing, model design, training, validation, and final evaluation.

Materials:

* [Notebook](Challenge-2/Challenge_2.ipynb)
* [Report](Challenge-2/ANNDL_report_challenge_2.pdf)
